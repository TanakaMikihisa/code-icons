# Code Icons

[EN](./README.md) / [JP](./README-JP.md) / [CH](./README-CH.md)

Code Icons は、Visual Studio Code / Cursor 向けのファイルアイコンテーマです。  
Seti の既存カバレッジを活かしつつ、よく使うファイル名・フォルダ名に独自アイコンを追加しています。

## 使い方

1. この拡張機能をインストール
2. コマンドパレットを開く
3. `File Icon Theme` を実行
4. `Code Icons` を選択
